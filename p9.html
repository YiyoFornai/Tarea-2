Hosting y Dominio

Un hosting es el espacio donde se aloja una página web para que pueda ser visualizado por cualquier usuario a través de un dominio mientras navega por Internet. El hosting es, en definitiva, el uso y provecho que se hace de esos mismos servidores web con todos los recursos de los que disponen

Por otro lado, un dominio es la dirección por la cual se encuentra el sitio web. Es un nombre único y es el que se escribe en la barra de direcciones del navegador para ingresar al sitio
